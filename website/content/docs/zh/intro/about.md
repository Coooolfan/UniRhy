UniRhy（独一律）是一个**自托管的音乐流媒体平台**。它把一台你能掌控的服务器（家庭 NAS、VPS、内网主机皆可）变成你私人的音乐播放后端：管理资源、分发音频、在多设备间同步播放状态。

## 核心特性

- **完全自托管**：所有数据存储在你自己控制的服务器上，无任何外部服务依赖。
- **跨平台客户端**：Web、macOS、Windows、Android、iOS，统一播放体验。
- **多设备播放同步**：同一账号在多个设备上保持播放进度一致，离开/加入同步会话顺滑切换。
- **插件化扩展**：通过插件接入第三方元数据、转码、艺术家归一化等能力。
- **音乐资源管理**：扫描、组织、转码，支持本地与对象存储。

## 整体结构

UniRhy 的运行只需要两类东西：

- **一个服务端**：负责音乐库管理、播放分发与多端同步，附带一个数据库。
- **若干客户端**：浏览器直接打开，或安装到 macOS、Windows、Android、iOS。

服务端与客户端之间走标准的 HTTP 与 WebSocket，部署只要保证客户端能访问到服务端即可，家用 NAS、VPS、内网主机都行。

更复杂的能力（第三方元数据补全、艺术家名归一化等）通过**插件**接入，按需启用，不影响核心功能。

## 当前状态

UniRhy 处于 **beta** 阶段。播放、同步、库管理等核心链路已可用并经过端到端测试覆盖；部分高级能力仍在路线图中。详见 [GitHub Releases](https://github.com/Coooolfan/UniRhy/releases)。
