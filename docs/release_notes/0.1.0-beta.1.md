## feat

### 账号与系统
* 账号注册、登录、鉴权与会话管理，支持注册限制与系统初始化流程
* 个人中心与系统设置页，支持账号信息编辑与 AI 模型配置

### 存储与内容模型
* 多存储节点管理，支持文件系统与 OSS 两种 Provider
* 作品（Work）、专辑（Album）、歌单（Playlist）、艺术家（Artist）、曲目（Track）核心实体与关联建模
* 作品、专辑、歌单的完整管理界面，支持编辑、合并、拖拽排序与批量操作
* 全局搜索功能与搜索结果页

### 播放器与多设备同步
* 基于 HTTP Range 的媒体流式播放与音频播放器 UI
* 基于 WebSocket 的同账号多设备播放同步，含 NTP 时钟对齐与调试面板
* 服务端播放队列管理，播放队列与播放进度跨会话持久化
* 播放模式偏好、Media Session API 系统媒体控制、最后设备离线自动暂停

### 异步任务与 AI 能力
* 异步任务管理系统、任务监控界面与任务日志查询
* 文件扫描、音频元数据与封面提取、Opus 转码
* 歌词向量化、数据清洗与 AI 歌单生成，Embedding 支持 OpenAI 兼容请求格式

### 多端适配
* Web、桌面端（Tauri）与 iOS 移动端支持，含移动端响应式布局
* 媒体 URL 签名机制保障多端媒体访问安全

### 官网与文档
* 官网双语首页、博客系统与文档中心（首次启动、数据模型、存储节点等）
